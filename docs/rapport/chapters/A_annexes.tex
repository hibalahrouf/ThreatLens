\section{Exemple de Fichier docker-compose.yml}
L'extrait suivant montre la configuration Docker Compose utilisée pour orchestrer les services de la plateforme.

\begin{tcolorbox}[codebox, title={Extrait de \code{docker-compose.yml}}]
\begin{minted}[fontsize=\footnotesize]{yaml}
version: '3.8'

services:
  db:
    image: postgres:15
    environment:
      POSTGRES_USER: ${POSTGRES_USER}
      POSTGRES_PASSWORD: ${POSTGRES_PASSWORD}
      POSTGRES_DB: ${POSTGRES_DB}
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

  api:
    build:
      context: ./backend
      dockerfile: Dockerfile
    command: uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
    volumes:
      - ./backend:/app
    ports:
      - "8000:8000"
    depends_on:
      - db
      - redis

  worker:
    build:
      context: ./backend
      dockerfile: Dockerfile
    command: celery -A app.core.celery_app worker --loglevel=info
    volumes:
      - ./backend:/app
    depends_on:
      - redis
      - db
\end{minted}
\end{tcolorbox}

\section{Extrait du Code de Mapping MASVS}
La fonction suivante illustre comment les résultats bruts de MobSF sont convertis vers les catégories du MASVS v2 en utilisant des expressions régulières.

\begin{tcolorbox}[codebox, title={Extrait de \code{masvs\_mapper.py}}]
\begin{minted}[fontsize=\footnotesize]{python}
import re

def map_mobsf_to_masvs_v2(finding_title: str, finding_desc: str) -> str:
    combined_text = (finding_title + " " + finding_desc).lower()
    
    if re.search(r'(crypto|md5|sha1|des|aes|rsa|mac|hash|random)', combined_text):
        return "MASVS-CRYPTO"
    elif re.search(r'(storage|sqlite|sharedpreference|sdcard|external|backup|cache)', combined_text):
        return "MASVS-STORAGE"
    elif re.search(r'(network|ssl|tls|http:|cleartext|certificate|pinning)', combined_text):
        return "MASVS-NETWORK"
    elif re.search(r'(auth|login|password|token|session|biometric)', combined_text):
        return "MASVS-AUTH"
    elif re.search(r'(intent|broadcast|content provider|service|ipc|webview)', combined_text):
        return "MASVS-PLATFORM"
    elif re.search(r'(obfuscation|root|jailbreak|debugger|tamper|frida)', combined_text):
        return "MASVS-RESILIENCE"
    else:
        return "MASVS-CODE"
\end{minted}
\end{tcolorbox}
