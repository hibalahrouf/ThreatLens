\section*{\textcolor{PrimaryBlue}{Résumé}}

Dans un contexte où les applications mobiles traitent des données de plus en plus
sensibles --- authentification biométrique, paiements en ligne, données médicales ---
la sécurité mobile est devenue un enjeu critique pour les organisations.
Pourtant, les audits de sécurité restent coûteux, longs et nécessitent une expertise
spécialisée rare.

\textbf{MASVS Audit Copilot} est une plateforme logicielle complète qui automatise
l'audit de sécurité des applications mobiles Android et iOS. Elle orchestrate plusieurs
moteurs d'analyse : l'analyse statique avancée via \textbf{MobSF}, la détection de
vulnérabilités dans les dépendances via \textbf{OSV.dev}, la cartographie des résultats
sur le standard \textbf{OWASP MASVS v2}, le scoring selon \textbf{CVSS v3.1}, et enfin
une couche d'intelligence artificielle locale (\textbf{Ollama / Llama 3}) pour le
triage automatisé des faux positifs et la génération de correctifs de code sécurisé.

La plateforme propose une interface web moderne (Next.js 15), une API RESTful
(FastAPI), une interface en ligne de commande complète (CLI Typer), et une génération
de rapports professionnels en formats PDF, Markdown et SARIF 2.1, adaptés aux
pipelines DevSecOps.

  LLM local, Ollama, FastAPI, Next.js, Docker, CVSS v3.1, SBOM, DevSecOps.

\section*{\textcolor{PrimaryBlue}{Abstract}}

In a context where mobile applications handle increasingly sensitive data ---
biometric authentication, online payments, medical records --- mobile security
has become a critical challenge for organizations. Yet, security audits remain
expensive, time-consuming, and require rare specialized expertise.

\textbf{MASVS Audit Copilot} is a comprehensive software platform that automates
the security audit of Android and iOS mobile applications. It orchestrates several
analysis engines: advanced static analysis via \textbf{MobSF}, dependency vulnerability
detection through \textbf{OSV.dev}, mapping of findings to the \textbf{OWASP MASVS v2}
standard, scoring according to \textbf{CVSS v3.1}, and finally a local artificial
intelligence layer (\textbf{Ollama / Llama 3}) for automated false-positive triage
and secure code patch generation.

The platform provides a modern web interface (Next.js 15), a RESTful API (FastAPI),
a full-featured command-line interface (Typer CLI), and professional report generation
in PDF, Markdown, and SARIF 2.1 formats, suitable for DevSecOps pipelines.
